# SpeedX Site Reset

A destructive admin-only WordPress reset plugin.

## Install (GitHub ZIP)

If you upload this repository ZIP directly in **Plugins → Add New → Upload Plugin**, WordPress expects the main plugin file to exist at the top level of the ZIP folder.

This repository is structured that way:

- `speedx-site-reset.php` (main plugin file)
- `uninstall.php`
- `assets/admin.css`
- `assets/admin.js`

So the GitHub-generated ZIP is upload-ready.

## Alternative install

You can also create your own ZIP named `speedx-site-reset.zip` containing these files in a top-level folder.
